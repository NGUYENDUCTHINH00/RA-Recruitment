        NOKA Devops Engineer (remote)

        Công ty Cổ phần Devwork

        Tất cả địa điểm

        Kỹ năng

        DevOpsAzureAWS

        Mô tả công việc

        Use Gitlab to auto-deploy our code branches it to a kubernetes cluster
        Add security monitoring like IDS to the cluster
        Implement integrations requested by customers
        Deploy updates and fixes
        Build tools to reduce occurrences of errors and improve customer experience
        Develop software to integrate with internal back-end systems
        Perform root cause analysis for production errors
        Investigate and resolve technical issues
        Develop scripts to automate visualization
        Design procedures for system troubleshooting and maintenance

        Yêu cầu công việc

        The person should have the following skill set:
        Deep knowledge of APIGEE OPDK API Gateway, or senior expertise on other API gateways
        Willingness to adopt to or gain Apigee OPDK knowledge, if not a given
        Architecture/Design new environments according to business needs
        Drive architecture/design evolution of existing environments
        AWS Knowledge
        Network and integrations: DNS, connectivity, VPC peering, Endpoints, VPNs, etc.
        Load balancing
        EC2
        Autoscaling groups
        Security, Secrets manager
        Cloudformation, OpsWorks, Chef
        Preferably coding skills: Python, troposphere to create Cloudformation templates
        Deep knowledge of Cassandra and related design and implementation patterns
        CI/CD and Automation Knowledge
        Jenkins
        Terraform
        Ansible
        HTTP Server / reverse Proxies
        Apache
        Squid
        Linux
        Centos / Redhat
        Ubuntu /Debian
        Shell scripting
        Kubernetes knowledge is an advantage
        Technologies and Skills, items with *** are a must have, ** should have:
        *** Network Architecture
        -- AWS Cloud and Data Center Architecture
        -- Virtual Private Network (VPN) Design and Implementation
        -- Transit Gateway
        -- AWS VPC Peering and Private Link
        ** AWS Cloud
        -- AWS EC2
        -- AWS Cli and Terraform
        ** Kubernetes
        -- GiantSwarm and Amazon EKS
        *** Kafka
        -- Kafka Cluster
        -- Topics, Users, ACLs, SASL/SCRAM Authentication
        -- Kafka REST Proxy
        -- Integration Architecture based on Private Link
        ** Certificate Management Automation for SSL/TLS Communication
        ** HAProxy and Squid
        *** CI/CD
        -- Github, Jenkins, user/service token
        *** Ansible
        -- Inventory and Playbook for resources deployment and configuration
        -- AWX Tower
        *** Logging Data Management
        -- Splunk and Cribl
        -- GCP Pub/Sub and Big Query
        -- AWS S3 Bucket and GCP Storage
        -- Kafka Connectors
        -- Logstash It's primarily about properly configuring caching of ORT artifacts on GitLab and docker/proxy issues. So DevOps / infrastructure issues

        - Opportunity to implement ambitious projects in many countries, exposure to the latest technologies and learn from good colleagues.
        - Working in a young, vibrant, modern and multicultural environment; Communication activities and events on holidays take place regularly.
        - Opportunity for advancement based on ability with corresponding increase in rank and salary increase.
        - Have the right to participate in soft skills training courses (logical thinking, creative thinking, communication skills, project management skills, negotiation skills…)
        - And many other attractive benefits...
