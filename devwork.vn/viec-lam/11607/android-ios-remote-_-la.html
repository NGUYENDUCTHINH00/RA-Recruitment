        Yêu cầu công việc

        The application should:
        1. Implement a login system (email / password) based on the available API1.
        2. No sign-up is required for this version.
        3. Once logged in, the merchant must enter the amount they wish to collect (using the
        same entry method as a classic TPE, cents first).
        4. A payment journey then begins by presenting a QR Code:
        This QR Code must contain a link2 that allows the Green Nation application to open
        on the client's phone by passing:
        - The amount of the operation
        - The currency
        - The external merchant identifier (provided by the API)
        - The merchant's identification token (provided by the API)
        - Encrypted with a private key of the merchant (provided by the API)
        Not to be carried out by the service provider, for information purposes:
        If the application is not installed, the link will display a simple web page to the
        user explaining the interest and role of Green Net Pay. This URL will be
        managed by Green Nation. There is no work to be done here.
        If the application is installed, the user must authenticate with FaceID /
        TouchID / Secret code to validate the payment. The payment is then validated
        and a transfer is made within the Green Nation application.
        5. When the QR Code is scanned by the client, the merchant's application must display
        "Payment in progress.
        6. Once the payment is finalized or canceled by the client, the merchant is notified4 of
        the result
